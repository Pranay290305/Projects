STM32 DISPLAY

STM32 Display a custom PCB for a Baja SAE dashboard display using STM32, CAN communication, and DC-DC converters, displaying SOC, speed, voltage,
battery temperature, motor temperature, motor controller temperature, vehicle mode, and motor RPM for reliable real-time visualization.
