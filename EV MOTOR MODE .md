TEMPERATURE BASED  EV MOTOR MODE SYSTEM
Temperature-based EV motor mode system (Eco, Boost, Idle) using ESP32 to optimize performance and thermal safety. Integrated temperature sensing, DC-DC power regulation,
and an OLED display for real-time speed, temperature, and mode monitoring.
